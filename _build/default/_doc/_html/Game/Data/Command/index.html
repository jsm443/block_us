Module Data.Command

type object_phrase = string list

The type object_phrase represents the object phrase that can be part of a player command. Each element of the list represents a word of the object phrase, where a word is defined as a consecutive sequence of non-space characters. Thus, no element of the list should contain any leading, internal, or trailing spaces. The list is in the same order as the words in the original player command.

type command =
| Place of object_phrase
| Quit
| Done
| See of string
| Malformed

The type command represents a player command that is decomposed into a verb and possibly an object phrase.

val parse : string -> command

parse str parses a player's input into a command, as follows. The first word (i.e., consecutive sequence of non-space characters) of str becomes the verb. The rest of the words, if any, become the object phrase.

Requires: str contains only alphanumeric (A-Z, a-z, 0-9) and space characters (only ASCII character code 32; not tabs or newlines, etc.).

Raises: Malformed if the command is malformed.
